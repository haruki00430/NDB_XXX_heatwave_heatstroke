# 参考文献の手動検証リスト

## 検証方法

各文献について、以下の手順で実在性を確認してください：

1. **PubMed** (https://pubmed.ncbi.nlm.nih.gov/) で検索
   - タイトルの一部 + 著者名で検索
   - 見つからない場合は次へ

2. **医中誌Web** (https://login.jamas.or.jp/) で検索
   - 日本語文献の場合

3. **Google Scholar** (https://scholar.google.com/) で検索
   - タイトルの一部 + 著者名で検索

4. **確認結果**
   - ✅ 実在確認 → チェックボックスにチェック
   - ❌ 実在しない → チェックボックスを空欄にし、コメント欄に「実在しない」と記入
   - ⚠️ 不明 → コメント欄に「要確認」と記入
