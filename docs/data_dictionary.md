## 算出変数の詳細

### 猛暑日数（Heatwave Days）

**定義**: 日最高気温が35℃以上の日数

**計算式**:
```python
heatwave_days = (df['気温最高'] >= 35.0).sum()
```

**根拠**: 気象庁の定義（猛暑日 = 日最高気温35℃以上）

---

### WBGT指数（暑さ指数）

**定義**: Wet Bulb Globe Temperature（湿球黒球温度）

**計算式**（屋外簡易版）:
```python
# 湿球温度の推定
wet_bulb_temp = dry_temp - (100 - humidity) / 5

# WBGT指数
wbgt = 0.7 * wet_bulb_temp + 0.3 * dry_temp
```

**基準値**:
- **31℃以上**: 危険（運動は原則中止）
- **28-31℃**: 厳重警戒（激しい運動は中止）
- **25-28℃**: 警戒（積極的に休憩）
- **25℃未満**: 注意（こまめに水分補給）

**根拠**: 環境省「熱中症予防情報サイト」

---

### 熱中症診療行為算定回数/10万人

**定義**: 熱中症関連の診療行為（救急医療管理加算・輸液）の人口10万人あたり算定回数

**計算式**:
```python
heatstroke_rate = (rescue_medical + infusion) / population * 100000
```

**根拠**: NDBオープンデータの標準的な標準化手法

## 欠損値処理

### 方針
1. **気象データ**: 欠損日は除外（欠損率<5%の場合）
2. **NDBデータ**: マスク値（"-"）はNaNに変換後、0として扱う（算定回数ゼロと解釈）
3. **e-Statデータ**: 欠損値がある都道府県は解析から除外

### 確認手順
```python
# 欠損値確認
df.isna().sum()

# 欠損率
df.isna().mean() * 100
```

## データ品質チェック

### 異常値の範囲

| 変数 | 下限 | 上限 | 理由 |
|------|------|------|------|
| `気温最高` | 20℃ | 45℃ | 夏季の日本の気温範囲 |
| `heatwave_days` | 0日 | 122日 | 対象期間の最大日数 |
| `heatstroke_rate` | 0件/10万人 | 1000件/10万人 | 生理学的に妥当な範囲 |
| `elderly_solo_rate` | 5% | 25% | 国勢調査の実績値範囲 |

# 異常値チェック
