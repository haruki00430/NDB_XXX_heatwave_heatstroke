# Reproduction Guide / 再現手順書

This document provides step-by-step instructions to reproduce all numerical results and figures reported in the manuscript.

**Note**: This repository does not include NDB raw data or intermediate data files (CSV/Parquet).  
- Input data: download from official sources listed in [README.md](README.md#data-sources--データソース).  
- Analysis outputs (figures): regenerated by running the pipeline steps below.

本書は論文に報告した全数値結果および図を再現するための手順書です。

**注意**: 本リポジトリは NDB 生データおよび中間ファイル（CSV/Parquet）を含みません。  
- 入力データ: [README.md](README.md) のデータソース一覧を参照し、各公式サイトからダウンロードしてください。  
- 解析出力（図）: 以下のパイプラインを実行すると再生成されます。

## Step 0: Environment Setup / 環境構築

```bash
git clone https://github.com/haruki00430/NDB_XXX_heatwave_heatstroke.git
cd NDB_XXX_heatwave_heatstroke

python -m venv .venv

# Windows
.venv\Scripts\activate
# macOS / Linux
source .venv/bin/activate

pip install -r requirements.txt
```

## Step 1: Data Preparation / データ準備

Place downloaded data in the directories shown below.

### 1-a. NDB Open Data No.10 (MHLW) — Outcome variable

```
02_Data/raw/NDB_OpenData/No.10/
```

Download from: https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/0000177182.html  
File required: infusion therapy (procedure code G004) prefecture-level aggregate table from the 10th NDB Open Data.

### 1-b. National Census 2020 (Statistics Bureau) — Exposure variable

```
02_Data/raw/Statistics_Bureau/Census_2020/
```

Files required: `tblT001141H01.txt` through `tblT001141H47.txt` (one file per prefecture)  
Download from: https://www.e-stat.go.jp/

### 1-c. JMA Weather Data — Climatic variables

JMA station data is downloaded automatically by ETL script 01 (requires internet access).

### 1-d. Air conditioning prevalence (MIC) — Covariate

```
02_Data/raw/MIC/NSFIE_2014/
```

Download from: https://www.stat.go.jp/data/zensho/2014/index.html

### 1-e. FDMA Heatstroke Transport (external validation only)

```
02_Data/raw/FDMA/
```

Download from: https://www.fdma.go.jp/disaster/heatstroke/post4.html  
File: summer 2023 daily heatstroke ambulance transport data by prefecture.

### 1-f. R5 Patient Survey (negative control only)

```
02_Data/raw/MHLW/Patient_Survey_R5/
```

Download from: https://www.mhlw.go.jp/toukei/list/10-20.html

---

## Step 2: ETL Pipeline / データ前処理

Run ETL scripts in order. Each script outputs intermediate files to `02_Data/interim/`.

```bash
python 03_Analysis/etl/01_jma_weather_data_download.py
python 03_Analysis/etl/02_process_jma_data.py
python 03_Analysis/etl/03_aggregate_weather_data.py
python 03_Analysis/etl/04_extract_elderly_solo_rate.py
```

| Script | Output | Description |
|--------|--------|-------------|
| `01_jma_weather_data_download.py` | `02_Data/interim/jma_raw/` | JMA station CSV download |
| `02_process_jma_data.py` | `02_Data/interim/jma_processed/` | Parse and clean JMA data |
| `03_aggregate_weather_data.py` | `02_Data/interim/jma_weather_prefecture.csv` | Prefecture-level aggregation |
| `04_extract_elderly_solo_rate.py` | `02_Data/interim/elderly_solo_household_rate.csv` | Census extraction |

## Step 3: Statistical Analysis / 統計解析

Run analysis scripts 01 through 09 in order.

```bash
python 03_Analysis/analysis/01_integrate_and_analyze.py

python 03_Analysis/analysis/07_additional_climate_variables.py
```
